About Chomee

Chomee is a software platform dedicated to educating to children in rural Africa. We accomplish this through students downloading lessons directly onto their mobile phones in an area with internet, then being able to complete the lesson offline where they do not have access to internet.

What I do as a Junior Full Stack Engineer for Chomee

  • Collaborate with the agile development team to design and implement features for the production app using git flow.
  • Champion backend development initiatives by harnessing the robust capabilities of Remix, delivering faster and more efficient serving of both dynamic and static content.
  • Shape the development landscape of our mobile client through impactful contributions using React Native for cross platform compatibility and performance.
  • Craft pristine, maintainable, and efficient TypeScript code, emphasizing a commitment to code quality and long-term sustainability.
  • Contribute actively to code reviews, upholding high-quality standards, while demonstrating expertise in troubleshooting and debugging to optimize application performance.
  • Continuously enhance professional expertise by staying abreast of the latest trends and advancements in development.
